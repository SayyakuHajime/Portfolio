01 · CNN Backpropagation

Backpropagation on CNN

Extending the gradient descent algorithm to update kernels and weights in a convolutional architecture. This involves propagating gradients through pooling and convolutional layers using the chain rule.

1.1 Forward Pass Recap

Before understanding the backward pass, we must define the computational flow of a CNN layer:

Convolution Layer
\(net_c = X * K\). The linear operation between input \(X\) and kernel \(K\).
Detector Layer
\(H = ReLU(net_c)\). Applies non-linearity, typically zeroing out negative values.
Pooling Layer
Reduksi dimensi (Max/Average) untuk efisiensi komputasi dan mencegah overfitting.
Fully Connected (FC)
\(O = Softmax(H \cdot W + b)\). The final classification layer.

1.2 Backward Mechanism

Gradien dihitung menggunakan Chain Rule untuk memperbarui parameter di setiap tahapan:

Update Weights FC (W)
\[ \frac{\partial E}{\partial w_{ji}} = \frac{\partial E}{\partial o_j} \cdot \frac{\partial o_j}{\partial net_j} \cdot \frac{\partial net_j}{\partial w_{ji}} \] \[ \Delta W = -(target - output) \cdot Output(1-Output) \cdot H \]
Standard backpropagation applied to the dense layers at the end of the network.
Update Kernel Konvolusi (K)
\[ \frac{\partial E}{\partial K} = \frac{\partial E}{\partial net_j} \cdot \frac{\partial net_j}{\partial H} \cdot \frac{\partial H}{\partial net_c} \cdot \frac{\partial net_c}{\partial K} \]
Gradien harus dirambatkan mundur melewati Pooling, Aktivasi (ReLU), dan akhirnya ke parameter Kernel.
Notes on Pooling: Pada Max Pooling, gradien hanya diteruskan ke unit yang memiliki nilai maksimum saat forward pass. Unit lainnya menerima gradien nol.
Parameter Counting with Weight Sharing
\[ \text{Params} = (\text{FilterWidth} \times \text{FilterHeight} \times \text{Channels} + 1) \times \text{NbFilters} \]
Weight sharing allows CNNs to be significantly more efficient compared to standard FFNNs.
02 · Recurrent Neural Networks

Recurrent Neural Networks (RNN)

A class of neural networks designed for sequential data where the order of information is crucial. Unlike FFNNs, RNNs maintain a internal state (memory) to process arbitrary sequences of inputs.

2.1 Motivation & Unfolding

Standard FFNNs assume inputs are independent. RNNs introduce Recurrent Connections to handle Context. Through Unfolding, we can represent the loop as a chain of units sharing the same parameters.

RNN Unrolling Visualization

2.2 Forward Propagation Equations

Hidden State (hₜ)
\[ h_t = f(Ux_t + Wh_{t-1} + b_{xh}) \]
\(U\): Weight matrix input → hidden  ·  \(W\): Weight matrix hidden → hidden (Recurrent)  ·  \(f\): Biasanya fungsi \(tanh\).
Output (yₜ)
\[ y_t = f(Vh_t + b_{hy}) \]
\(V\): Weight matrix hidden → output  ·  \(f\): Softmax (klasifikasi) atau Linear (regresi).
RNN Cell — Internal Structure

2.3 Architectural Variations

One-to-Many
Single input, sequence output (e.g., Image Captioning).
Many-to-One
Sequence input, single output (e.g., Sentiment Analysis).
Many-to-Many (Sync)
Input length = Output length (e.g., POS Tagging).
Many-to-Many (Seq2Seq)
Encoder-Decoder pattern (e.g., Machine Translation).
Bidirectional RNN (BRNN): Memiliki dua hidden layer (maju dan mundur). Output \(y_t\) bergantung pada informasi masa lalu sekaligus masa depan dalam sequence tersebut.
03 · Long Short-Term Memory

LSTM & Vanishing Gradient

LSTM was introduced to solve the Vanishing Gradient problem in standard RNNs, allowing the network to learn long-range dependencies by maintaining a constant error flow through the Cell State.

3.1 The Vanishing Gradient Problem

Pada RNN standar, gradien dikalikan berulang kali dengan angka kecil (turunan \(tanh\) < 1) saat backpropagation. Hal ini menyebabkan gradien mengecil secara eksponensial, sehingga model tidak bisa mempelajari ketergantungan jangka panjang.

3.2 LSTM Structure & Gates

LSTM menggunakan mekanisme Gates untuk mengatur aliran informasi:

LSTM Cell — Gate Structure & Cell State Highway
The Four Gates Logic
\[ f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f) \quad \text{(Forget)} \] \[ i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i) \quad \text{(Input)} \] \[ \tilde{C}_t = tanh(W_c \cdot [h_{t-1}, x_t] + b_c) \quad \text{(Candidate)} \] \[ o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o) \quad \text{(Output)} \]
State Updates
\[ C_t = f_t * C_{t-1} + i_t * \tilde{C}_t \] \[ h_t = o_t * tanh(C_t) \]
Informasi masa lalu difilter oleh forget gate, informasi baru ditambahkan oleh input gate.
04 · Backpropagation Through Time

BPTT Algorithm

BPTT is the standard algorithm for training recurrent networks by unrolling them into a deep feedforward network where each layer shares the same weights.

BPTT Procedure
1. ForwardHitung semua \(h_t\) dan \(y_t\) untuk satu urutan penuh (sequence).
2. ErrorHitung total loss \(E = \sum E_t\) (biasanya menggunakan Cross-Entropy).
3. BackwardKalkulasi gradien pada setiap timestep dan akumulasikan karena bobot \(U, V, W\) bersifat global.
Truncated BPTT: Untuk urutan yang sangat panjang, proses backward dibatasi hanya beberapa langkah ke belakang untuk menghemat memori dan mencegah komputasi berlebih.
Glossary

Sequential Modeling Terms

Key definitions for Final Term material.

Sequence Data in which the order of elements matters.
Recurrent Connection Feedback loop that feeds hidden state back to the same unit.
Cell State The long-term memory component of an LSTM cell.
Vanishing Gradient When gradients become too small to update early weights.
Unrolling Representing an RNN as a deep chain of units.
Parameter Sharing Using the same weight matrices across all timesteps.
