\[ y = f(\mathbf{x}; \mathbf{w}) = \mathbf{w}^T \mathbf{x} + b \]
Briefly explain every symbol: \(\mathbf{w}\) = weight vector  ·  \(\mathbf{x}\) = input  ·  \(b\) = bias.
Error / Loss Function
\[ E(\mathbf{w}) = \frac{1}{2}\sum_{d \in D}(y_d - o_d)^2 \]
\(o_d\) = predicted output for example \(d\)  ·  \(y_d\) = true label. The \(\tfrac{1}{2}\) simplifies the derivative.

2.1 Comparison

Approach A
\[ \Delta w_k = \eta\sum_{d \in D}(y_d - o_d)(x_{dk}) \]

One weight update per epoch. Stable but slow for large datasets.

Approach B
\[ \Delta w_k = \eta\,(y_d - o_d)(x_{dk}) \]

One update per example. Faster and noisier — can escape local minima.
