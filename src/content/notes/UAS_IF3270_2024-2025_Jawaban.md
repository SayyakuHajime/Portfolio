### Soal 1

**Soal:** Terdapat spesifikasi arsitektur neural network **"many-to-one"** yang menerima input terdiri atas **3 fitur**, sebuah hidden layer berupa RNN yang terdiri atas **4 neuron** dengan **5 time-step**, dan sebuah output layer yang terdiri atas **dua neuron**. **Analisis setiap pilihan (berdasarkan gambar soal):**

- **a** — 5 blok `h(4)`, setiap blok menerima `x(3)`, output `y1` & `y2` muncul dari **dua timestep terakhir** (t=4 dan t=5).
  → **X** ❌ — many-to-one seharusnya output **hanya dari 1 timestep terakhir saja**, bukan dua.

- **b** — 5 blok `h(4)`, tetapi input-nya adalah "Fitur 1 x1", "Fitur 2 x1", "Fitur 3 x1" (masing-masing **1 fitur per timestep berbeda**), output `y1` & `y2` dari dua timestep terakhir.
  → **X** ❌ — Interpretasi fitur salah: seharusnya tiap timestep menerima **vektor 3 fitur** ($x(3)$), bukan 1 fitur per timestep. Lagi pula output dari 2 timestep.

- **c** — Hanya **3 blok** `h(4)`, input `x(3)` tiap timestep, output `y(2)` dari timestep terakhir saja.
  → **X** ❌ — Jumlah timestep hanya 3, seharusnya **5**.

- **d** — 4 blok (`h1, h2, h3, h4`) dengan input masing-masing "Fitur 1", "Fitur 2", "Fitur 3" sebagai 3 timestep terpisah, output `y(2)` dari timestep terakhir.
  → **X** ❌ — 3 fitur diinterpretasikan sebagai 3 timestep (bukan vektor fitur dalam 1 timestep), dan hanya 4 blok hidden. Tidak sesuai.

- **e** — **5 blok** `h(4)`, setiap blok menerima `x(3)` (vektor 3 fitur), output `y(2)` **hanya dari timestep t=5** (terakhir) via bobot V.
  → **O** ✅ — Sesuai spesifikasi: 5 timestep, 4 neuron hidden, 3 fitur input, output hanya dari timestep terakhir = many-to-one.

### Soal 2

> [!note] Kunci Interpretasi

**a.** *Untuk t=1, maka data inputnya adalah: ⟨0.1, 0.2, 0.3⟩.*
→ **O** ✅ — Baris pertama, kolom x1 x2 x3.

**b.** *Untuk t=2, maka data inputnya adalah: ⟨0.2, 0.5, 0.8⟩.*
→ **X** ❌ — Input t=2 adalah baris kedua = **⟨0.4, 0.5, 0.6⟩**. Pilihan ⟨0.2, 0.5, 0.8⟩ adalah nilai kolom x1 di t=1,2,3 dibaca ke bawah (interpretasi columnar yang salah untuk many-to-many equal).

**c.** *Nilai neuron pertama di hidden layer t=2 adalah:* $\tanh(0.4 \times 0.1 + 0.5 \times 0.1 + 0.6 \times 0.1 + 0.2 \times x + 0.1)$
→ **O** ✅ — Input t=2 = ⟨0.4, 0.5, 0.6⟩, tiap fitur × bobot 0.1, plus kontribusi hidden sebelumnya $0.2 \times x$, plus bias 0.1.

**d.** *Nilai neuron pertama di hidden layer t=2 adalah:* $\tanh(0.2 \times 0.1 + 0.5 \times 0.1 + 0.8 \times 0.1 + 0.2 \times x + 0.1)$
→ **X** ❌ — Nilai 0.2 dan 0.8 adalah data kolom x1 di t=1 dan t=3, bukan input t=2 yang benar ⟨0.4, 0.5, 0.6⟩.

**e.** *Jika nilai hidden t=2 adalah $z = [z_1, z_2]$, nilai output t=2 adalah:* $\text{sigmoid}(0.3 \times z_1 + 0.3 \times z_2 + 0.3)$
→ **O** ✅ — 2 hidden neuron masing-masing berbobot 0.3, bias 0.3, ke 1 neuron output.

### Soal 3

**Soal:** Input 10 fitur. Hidden 1: Simple RNN 6 neuron. Hidden 2: Simple RNN 5 neuron. Hidden 3: Simple RNN 4 neuron. Hidden 4: Dense 3 neuron. Output: Dense 2 neuron. Banyaknya parameter?

> [!note] Formula Jumlah Parameter
> **Simple RNN** dengan $n$ neuron, menerima input dari dimensi $m$:
> $$\text{params} = \underbrace{n \times m}_{W_{xh}} + \underbrace{n \times n}_{W_{hh}} + \underbrace{n}_{b_h} = (m + n + 1) \times n$$
>
> **Dense layer** dengan $k$ neuron, input dari dimensi $m$:
> $$\text{params} = (m + 1) \times k$$

**Layer 1 — Simple RNN, 6 neuron, input $m=10$:**
$$\text{params}_1 = (m + n + 1) \times n = (10 + 6 + 1) \times 6 = 17 \times 6 = 102$$

**Layer 2 — Simple RNN, 5 neuron, input dari 6 neuron ($m=6$):**
$$\text{params}_2 = (m + n + 1) \times n = (6 + 5 + 1) \times 5 = 12 \times 5 = 60$$

**Layer 3 — Simple RNN, 4 neuron, input dari 5 neuron ($m=5$):**
$$\text{params}_3 = (m + n + 1) \times n = (5 + 4 + 1) \times 4 = 10 \times 4 = 40$$

**Layer 4 — Dense, 3 neuron, input dari 4 neuron ($m=4$):**
$$\text{params}_4 = (m + 1) \times k = (4 + 1) \times 3 = 5 \times 3 = 15$$

**Output Layer — Dense, 2 neuron, input dari 3 neuron ($m=3$):**
$$\text{params}_5 = (m + 1) \times k = (3 + 1) \times 2 = 4 \times 2 = 8$$

**Total:**
$$\text{Total} = 102 + 60 + 40 + 15 + 8 = \boxed{225}$$

**Jawaban: b. 225** ✅

## Bagian II — LSTM Forward & Backward Pass (Nilai 20)

### Setup Soal

**Soal:** LSTM sederhana, 1 hidden neuron ($h \in \mathbb{R}^1$), input 2 fitur ($x \in \mathbb{R}^2$). Target adalah nilai pada hidden unit.

**Matriks Input:**

| id  | X1  | X2  | Target |
|-----|-----|-----|--------|
| id1 | 0.5 | 1.2 | 0.5    |
| id2 | 1.5 | 1.0 | 0.75   |

→ id1 = timestep $t=1$, id2 = timestep $t=2$.

**Initial state:** $h_0 = 0$, $c_0 = 0$

**Matriks Weight:**

| Gate | $U$ (input→hidden) | $W$ (hidden→hidden) | $b$ (bias) |
|------|---------------------|----------------------|-----------|
| Forget $f$ | $U_f = [1,\ 1]$ | $W_f = 0.5$ | $b_f = 1$ |
| Input $i$ | $U_i = [1,\ 0]$ | $W_i = 0.3$ | $b_i = -1$ |
| Candidate $\tilde{c}$ | $U_c = [0,\ 0]$ | $W_c = 0.5$ | $b_c = 1$ |
| Output $o$ | $U_o = [0,\ 1]$ | $W_o = 0.3$ | $b_o = -1$ |

> [!note] Fungsi Aktivasi LSTM Default
> - Gate (forget, input, output): **sigmoid** $\sigma(z) = \dfrac{1}{1+e^{-z}}$
> - Candidate cell state: **tanh**
> - Hidden output: $h_t = o_t \cdot \tanh(c_t)$

**Formula LSTM (notasi skalar, 1 neuron):**

$$f_t = \sigma(U_f \cdot x_t + W_f \cdot h_{t-1} + b_f)$$
$$i_t = \sigma(U_i \cdot x_t + W_i \cdot h_{t-1} + b_i)$$
$$\tilde{c}_t = \tanh(U_c \cdot x_t + W_c \cdot h_{t-1} + b_c)$$
$$c_t = f_t \cdot c_{t-1} + i_t \cdot \tilde{c}_t$$
$$o_t = \sigma(U_o \cdot x_t + W_o \cdot h_{t-1} + b_o)$$
$$h_t = o_t \cdot \tanh(c_t)$$

---

### 2.a Forward Pass

#### Timestep $t=1$: $x_1 = [0.5,\ 1.2]$, $h_0=0$, $c_0=0$

**Forget gate $f_1$:**
$$z_{f_1} = U_f \cdot x_1 + W_f \cdot h_0 + b_f = (1)(0.5) + (1)(1.2) + (0.5)(0) + 1 = 0.5 + 1.2 + 0 + 1 = 2.7$$
$$f_1 = \sigma(2.7) = \frac{1}{1+e^{-2.7}} = \frac{1}{1+0.067} = \frac{1}{1.067} = \mathbf{0.937}$$

**Input gate $i_1$:**
$$z_{i_1} = U_i \cdot x_1 + W_i \cdot h_0 + b_i = (1)(0.5) + (0)(1.2) + (0.3)(0) + (-1) = 0.5 + 0 + 0 - 1 = -0.5$$
$$i_1 = \sigma(-0.5) = \frac{1}{1+e^{0.5}} = \frac{1}{1+1.649} = \frac{1}{2.649} = \mathbf{0.378}$$

**Candidate cell $\tilde{c}_1$:**
$$z_{\tilde{c}_1} = U_c \cdot x_1 + W_c \cdot h_0 + b_c = (0)(0.5) + (0)(1.2) + (0.5)(0) + 1 = 0 + 0 + 0 + 1 = 1.0$$
$$\tilde{c}_1 = \tanh(1.0) = \mathbf{0.762}$$

**Cell state $c_1$:**
$$c_1 = f_1 \cdot c_0 + i_1 \cdot \tilde{c}_1 = (0.937)(0) + (0.378)(0.762) = 0 + 0.288 = \mathbf{0.288}$$

**Output gate $o_1$:**
$$z_{o_1} = U_o \cdot x_1 + W_o \cdot h_0 + b_o = (0)(0.5) + (1)(1.2) + (0.3)(0) + (-1) = 0 + 1.2 + 0 - 1 = 0.2$$
$$o_1 = \sigma(0.2) = \frac{1}{1+e^{-0.2}} = \frac{1}{1+0.819} = \frac{1}{1.819} = \mathbf{0.550}$$

**Hidden state $h_1$:**
$$h_1 = o_1 \cdot \tanh(c_1) = (0.550) \cdot \tanh(0.288) = (0.550)(0.280) = \mathbf{0.154}$$

---

#### Timestep $t=2$: $x_2 = [1.5,\ 1.0]$, $h_1=0.154$, $c_1=0.288$

**Forget gate $f_2$:**
$$z_{f_2} = (1)(1.5) + (1)(1.0) + (0.5)(0.154) + 1 = 1.5 + 1.0 + 0.077 + 1 = 3.577$$
$$f_2 = \sigma(3.577) = \frac{1}{1+e^{-3.577}} = \frac{1}{1+0.028} = \mathbf{0.973}$$

**Input gate $i_2$:**
$$z_{i_2} = (1)(1.5) + (0)(1.0) + (0.3)(0.154) + (-1) = 1.5 + 0 + 0.046 - 1 = 0.546$$
$$i_2 = \sigma(0.546) = \frac{1}{1+e^{-0.546}} = \frac{1}{1+0.579} = \mathbf{0.633}$$

**Candidate cell $\tilde{c}_2$:**
$$z_{\tilde{c}_2} = (0)(1.5) + (0)(1.0) + (0.5)(0.154) + 1 = 0 + 0 + 0.077 + 1 = 1.077$$
$$\tilde{c}_2 = \tanh(1.077) = \mathbf{0.792}$$

**Cell state $c_2$:**
$$c_2 = f_2 \cdot c_1 + i_2 \cdot \tilde{c}_2 = (0.973)(0.288) + (0.633)(0.792) = 0.280 + 0.501 = \mathbf{0.781}$$

**Output gate $o_2$:**
$$z_{o_2} = (0)(1.5) + (1)(1.0) + (0.3)(0.154) + (-1) = 0 + 1.0 + 0.046 - 1 = 0.046$$
$$o_2 = \sigma(0.046) = \frac{1}{1+e^{-0.046}} = \frac{1}{1+0.955} = \mathbf{0.512}$$

**Hidden state $h_2$:**
$$h_2 = o_2 \cdot \tanh(c_2) = (0.512) \cdot \tanh(0.781) = (0.512)(0.653) = \mathbf{0.334}$$

---

#### Hitung Error Total

Target: $\hat{h}_1 = 0.5$, $\hat{h}_2 = 0.75$. Gunakan **MSE**: $\mathcal{L}_t = \frac{1}{2}(h_t - \hat{h}_t)^2$

$$\mathcal{L}_1 = \frac{1}{2}(h_1 - \hat{h}_1)^2 = \frac{1}{2}(0.154 - 0.5)^2 = \frac{1}{2}(-0.346)^2 = \frac{1}{2}(0.120) = \mathbf{0.060}$$

$$\mathcal{L}_2 = \frac{1}{2}(h_2 - \hat{h}_2)^2 = \frac{1}{2}(0.334 - 0.75)^2 = \frac{1}{2}(-0.416)^2 = \frac{1}{2}(0.173) = \mathbf{0.086}$$

$$\boxed{\mathcal{L}_{\text{total}} = \mathcal{L}_1 + \mathcal{L}_2 = 0.060 + 0.086 = 0.146}$$

**Ringkasan Forward Pass:**
| Variabel | t=1 | t=2 |
|----------|-----|-----|
| $f_t$ | 0.937 | 0.973 |
| $i_t$ | 0.378 | 0.633 |
| $\tilde{c}_t$ | 0.762 | 0.792 |
| $c_t$ | 0.288 | 0.781 |
| $o_t$ | 0.550 | 0.512 |
| $h_t$ | 0.154 | 0.334 |
| $\tanh(c_t)$ | 0.280 | 0.653 |

### 2.b Backward Pass (BPTT), $\eta = 0.5$

> [!note] Notasi & Turunan
> - $\sigma'(z) = \sigma(z)(1-\sigma(z))$, sehingga $\frac{\partial f_t}{\partial z_{f_t}} = f_t(1-f_t)$
> - $\tanh'(z) = 1 - \tanh^2(z)$, sehingga $\frac{\partial \tilde{c}_t}{\partial z_{\tilde{c}_t}} = 1-\tilde{c}_t^2$
> - Dari MSE: $\frac{\partial \mathcal{L}_t}{\partial h_t} = h_t - \hat{h}_t$
> - Update rule: $W \leftarrow W - \eta \cdot \frac{\partial \mathcal{L}}{\partial W}$

$$\frac{\partial \mathcal{L}}{\partial h_2} = h_2 - \hat{h}_2 = 0.334 - 0.750 = -0.416$$

## Bagian III — Encoder-Decoder & Attention (Nilai 25)

### Setup Soal

**Soal:** Encoder-Decoder untuk prediksi sekuens 2 elemen deret.
- **Encoder:** input sequence panjang **3 timestep**, vektor **4 fitur** → menghasilkan hidden vector $e_1, e_2, e_3$
- **Decoder:** menerima context vector → menghasilkan sequence **2 timestep** yaitu $y_1$ dan $y_2$

---

### 3.a Jumlah Parameter — Encoder RNN(2) + Decoder RNN(2) + FC(1)

> [!note] Formula Parameter
> **Simple RNN** ($n$ neuron, input $m$ dimensi):
> $$\text{params} = \underbrace{n \times m}_{U: \text{input→hidden}} + \underbrace{n \times n}_{W: \text{hidden→hidden}} + \underbrace{n}_{b} = (m+n+1) \times n$$
>
> **Dense/FC** ($k$ neuron, input $m$ dimensi):
> $$\text{params} = (m+1) \times k$$

**Encoder RNN:** $n=2$ neuron, input $m=4$ fitur
$$\text{params}_{\text{enc}} = (m+n+1) \times n = (4+2+1) \times 2 = 7 \times 2 = 14$$

**Decoder RNN:** $n=2$ neuron, input = context vector dari encoder = dimensi $m=2$
$$\text{params}_{\text{dec}} = (m+n+1) \times n = (2+2+1) \times 2 = 5 \times 2 = 10$$

**FC Output:** $k=1$ neuron, input dari hidden decoder $m=2$
$$\text{params}_{\text{FC}} = (m+1) \times k = (2+1) \times 1 = 3$$

$$\boxed{\text{Total} = 14 + 10 + 3 = 27}$$

---

### 3.b Jumlah Parameter — Encoder LSTM(2) + Decoder LSTM(2) + FC(1)

> [!note] Formula Parameter LSTM
> **LSTM** ($n$ neuron, input $m$ dimensi): terdapat **4 gate**, masing-masing punya set weight sendiri:
> $$\text{params} = (m+n+1) \times 4n$$

**Encoder LSTM:** $n=2$ neuron, input $m=4$ fitur
$$\text{params}_{\text{enc}} = (m+n+1) \times 4n = (4+2+1) \times 4 \times 2 = 7 \times 8 = 56$$

**Decoder LSTM:** $n=2$ neuron, input $m=2$ (dari context vector)
$$\text{params}_{\text{dec}} = (m+n+1) \times 4n = (2+2+1) \times 4 \times 2 = 5 \times 8 = 40$$

**FC Output:** $k=1$ neuron, input $m=2$
$$\text{params}_{\text{FC}} = (2+1) \times 1 = 3$$

$$\boxed{\text{Total} = 56 + 40 + 3 = 99}$$

---

### 3.c Formula Inferensi Decoder (RNN, tanpa Attention)

**Konteks:** Context vector $c = e_3$ (hidden state terakhir encoder) → initial hidden state decoder $s_0 = c$.

**Inisialisasi:**
$$s_0 = c = e_3$$

**Timestep $t=1$** — input decoder: $y_0$ (bisa token `<start>` atau context vector $c$):
$$s_1 = f(U_d \cdot y_0 + W_d \cdot s_0 + b_d)$$
$$y_1 = g(V \cdot s_1 + b_y)$$

**Timestep $t=2$** — input decoder: $y_1$ (output sebelumnya, autoregressive):
$$s_2 = f(U_d \cdot y_1 + W_d \cdot s_1 + b_d)$$
$$y_2 = g(V \cdot s_2 + b_y)$$

**Formula umum untuk timestep $t$:**

$$\boxed{s_t = f(U_d \cdot y_{t-1} + W_d \cdot s_{t-1} + b_d)}$$
$$\boxed{y_t = g(V \cdot s_t + b_y)}$$

dimana $f$ = fungsi aktivasi hidden layer decoder, $g$ = fungsi aktivasi output layer.

---

### 3.d Jumlah Parameter — Encoder RNN(2) + Decoder RNN(2) + FC(1) + 1 Attention Unit

**Dari soal a:** Total parameter (tanpa attention) = 27

**Parameter Attention Unit (Bahdanau-style):**

$$e_{tj} = \text{align}(s_{t-1}, h_j) = v_a^\top \tanh(W_a s_{t-1} + U_a h_j)$$

dimana:
- $W_a$: bobot untuk $s_{t-1}$ (decoder), ukuran $n_a \times n_{\text{dec}} = 2 \times 2 = 4$ parameter
- $U_a$: bobot untuk $h_j$ (encoder), ukuran $n_a \times n_{\text{enc}} = 2 \times 2 = 4$ parameter
- $v_a$: vektor proyeksi, ukuran $n_a = 2$ parameter

$$\text{params}_{\text{attention}} = 4 + 4 + 2 = 10$$

$$\boxed{\text{Total} = 27 + 10 = 37}$$

---

### 3.e Formula Inferensi Decoder dengan Attention ($\alpha_t$)

Dengan attention, context vector berbeda di **setiap timestep decoder** ($c_t$, bukan $c$ tunggal yang konstan).

**Hitung alignment score** antara $s_{t-1}$ dan setiap hidden encoder $h_j$ ($j=1,2,3$):
$$e_{tj} = \text{align}(s_{t-1}, h_j)$$

**Hitung attention weights** via softmax:
$$\alpha_{tj} = \frac{\exp(e_{tj})}{\sum_{k=1}^{3} \exp(e_{tk})}$$

**Hitung context vector** pada timestep $t$:
$$c_t = \sum_{j=1}^{3} \alpha_{tj} \cdot h_j$$

**Formula decoder dengan attention:**

$$\boxed{s_t = f(U_d \cdot y_{t-1} + W_d \cdot s_{t-1} + W_c \cdot c_t + b_d)}$$
$$\boxed{y_t = g(V \cdot s_t + b_y)}$$

**Lengkap untuk $t=1$ dan $t=2$:**

*Timestep t=1:*
$$e_{1j} = \text{align}(s_0, h_j), \quad j \in \{1,2,3\}$$
$$\alpha_{1j} = \frac{\exp(e_{1j})}{\sum_{k=1}^{3}\exp(e_{1k})}, \quad \sum_j \alpha_{1j} = 1$$
$$c_1 = \alpha_{11} h_1 + \alpha_{12} h_2 + \alpha_{13} h_3$$
$$s_1 = f(U_d \cdot y_0 + W_d \cdot s_0 + W_c \cdot c_1 + b_d)$$
$$y_1 = g(V \cdot s_1 + b_y)$$

*Timestep t=2:*
$$e_{2j} = \text{align}(s_1, h_j), \quad j \in \{1,2,3\}$$
$$\alpha_{2j} = \frac{\exp(e_{2j})}{\sum_{k=1}^{3}\exp(e_{2k})}$$
$$c_2 = \alpha_{21} h_1 + \alpha_{22} h_2 + \alpha_{23} h_3$$
$$s_2 = f(U_d \cdot y_1 + W_d \cdot s_1 + W_c \cdot c_2 + b_d)$$
$$y_2 = g(V \cdot s_2 + b_y)$$

## Bagian IV — Reinforcement Learning (Nilai 25)

### 4.1 Tabel Perbandingan Supervised Learning vs Reinforcement Learning (Nilai 6)

**Soal:** Lengkapi tabel perbandingan.

| Aspek | Supervised Learning | Reinforcement Learning |
|-------|---------------------|------------------------|
| **Informasi yang diperlukan oleh agen** | **Pasangan aksi yang harus dilakukan dan kondisi tertentu** *(labeled input-output pairs)* | **Informasi reward** *(sinyal reward dari lingkungan setelah tiap aksi)* |
| **Hasil observasi berupa data sekuensial** | **Belum Tentu** *(bisa i.i.d. seperti klasifikasi gambar, bisa sekuensial seperti time series)* | **Ya** *(agen berinteraksi dengan lingkungan secara sekuensial: state→action→reward→new state)* |
| **Apakah yang dipelajari oleh agen?** | **Fungsi pemetaan dari input ke output** *(mapping $f: X \to Y$)* | **Policy (kebijakan) terbaik** — yaitu aksi mana yang harus diambil pada setiap state agar cumulative reward maksimal |

### 4.2 Wumpus World — TD Q-Learning (Nilai 19)

#### Setup Grid (dari gambar soal)

Berdasarkan gambar soal (grid 4×4, koordinat $(x,y)$ dengan $x$ = kolom, $y$ = baris):

```
y=4 | Stench  |         | Breeze  | PIT     |
y=3 | WUMPUS  | Breeze  | PIT     | Breeze  |
y=2 | Stench  |         | Breeze  |         |
y=1 | START   | Breeze  | PIT     | Breeze  |
      x=1       x=2       x=3       x=4
```

**Terminal states:** Wumpus di **(3,1)**, Gold di **(3,2)**, Pit di **(1,3)** (dan kemungkinan (3,1) pit juga tapi soal menyatakan wumpus di (3,1)), Pit di **(4,4)** dan **(3,1)**.

> [!note] Catatan dari Gambar
> Dari gambar soal: Wumpus di (1,3) area (gambar menunjukkan wumpus di row 3 col 1), Gold di (3,2) bukan (3,3). Soal menyatakan: **wumpus di (3,1), gold di (3,2), pit di (1,3)**. Sesuai teks soal ini yang kita pakai.

**Reward:**
- Transisi normal (non-terminal): $R = 0$
- Menuju **Gold** (3,2): $R = +10$
- Menuju **Wumpus** (3,1) atau **Pit**: $R = -10$
- State terminal: Q-value = 0

**Parameter:** $\alpha = 0.4$, $\gamma = 0.6$, inisialisasi semua $Q(s,a) = 0$

**Formula Q-Learning (off-policy TD):**

$$Q(s, a) \leftarrow Q(s, a) + \alpha \Big[R + \gamma \max_{a'} Q(s', a') - Q(s, a)\Big]$$

---

#### (i) Update Nilai Q(s,a) per Episode

##### Episode I: $(1,1) \to (2,1) \to (3,1)$

**Step 1:** $(1,1) \xrightarrow{E} (2,1)$, $s=(1,1)$, $a=E$, $s'=(2,1)$

$R = 0$ (transisi normal), $\max_{a'} Q((2,1), a') = \max\{0,0,0,0\} = 0$

$$Q((1,1), E) \leftarrow Q((1,1), E) + \alpha\left[R + \gamma \max_{a'} Q((2,1), a') - Q((1,1), E)\right]$$
$$= 0 + 0.4\left[0 + 0.6 \cdot 0 - 0\right] = 0 + 0.4(0) = \mathbf{0}$$

**Step 2:** $(2,1) \xrightarrow{E} (3,1)$, $s=(2,1)$, $a=E$, $s'=(3,1)$ **[Terminal: Wumpus]**

$R = -10$, $\max_{a'} Q((3,1), a') = 0$ (terminal state)

$$Q((2,1), E) \leftarrow 0 + 0.4\left[-10 + 0.6 \cdot 0 - 0\right]$$
$$= 0 + 0.4 \cdot (-10) = \mathbf{-4}$$

---

##### Episode II: $(1,1) \to (1,2) \to (2,2) \to (3,2)$

**Step 1:** $(1,1) \xrightarrow{N} (1,2)$, $s=(1,1)$, $a=N$, $s'=(1,2)$

$R = 0$, $\max_{a'} Q((1,2), a') = \max\{0,0,0,0\} = 0$

$$Q((1,1), N) \leftarrow 0 + 0.4\left[0 + 0.6 \cdot 0 - 0\right] = \mathbf{0}$$

**Step 2:** $(1,2) \xrightarrow{E} (2,2)$, $s=(1,2)$, $a=E$, $s'=(2,2)$

$R = 0$, $\max_{a'} Q((2,2), a') = \max\{0,0,0,0\} = 0$

$$Q((1,2), E) \leftarrow 0 + 0.4\left[0 + 0.6 \cdot 0 - 0\right] = \mathbf{0}$$

**Step 3:** $(2,2) \xrightarrow{E} (3,2)$, $s=(2,2)$, $a=E$, $s'=(3,2)$ **[Terminal: Gold]**

$R = +10$, $\max_{a'} Q((3,2), a') = 0$ (terminal state)

$$Q((2,2), E) \leftarrow 0 + 0.4\left[10 + 0.6 \cdot 0 - 0\right]$$
$$= 0 + 0.4 \cdot 10 = \mathbf{+4}$$

---

##### Episode III: $(1,1) \to (1,2) \to (1,3)$

**Step 1:** $(1,1) \xrightarrow{N} (1,2)$, $s=(1,1)$, $a=N$, $s'=(1,2)$

$R = 0$, $\max_{a'} Q((1,2), a') = \max\{Q(N), Q(E), Q(S), Q(W)\} = \max\{0, 0, 0, 0\} = 0$

$$Q((1,1), N) \leftarrow 0 + 0.4\left[0 + 0.6 \cdot 0 - 0\right] = \mathbf{0}$$

**Step 2:** $(1,2) \xrightarrow{N} (1,3)$, $s=(1,2)$, $a=N$, $s'=(1,3)$ **[Terminal: Pit]**

$R = -10$, $\max_{a'} Q((1,3), a') = 0$ (terminal state)

$$Q((1,2), N) \leftarrow 0 + 0.4\left[-10 + 0.6 \cdot 0 - 0\right]$$
$$= 0 + 0.4 \cdot (-10) = \mathbf{-4}$$

**Ringkasan Semua Update:**

| Episode | Step | Transisi | R | Q sebelum | Q sesudah |
|---------|------|----------|---|-----------|-----------|
| I | 1 | $(1,1) \xrightarrow{E} (2,1)$ | 0 | 0 | **0** |
| I | 2 | $(2,1) \xrightarrow{E} (3,1)$★ | -10 | 0 | **-4** |
| II | 1 | $(1,1) \xrightarrow{N} (1,2)$ | 0 | 0 | **0** |
| II | 2 | $(1,2) \xrightarrow{E} (2,2)$ | 0 | 0 | **0** |
| II | 3 | $(2,2) \xrightarrow{E} (3,2)$★ | +10 | 0 | **+4** |
| III | 1 | $(1,1) \xrightarrow{N} (1,2)$ | 0 | 0 | **0** |
| III | 2 | $(1,2) \xrightarrow{N} (1,3)$★ | -10 | 0 | **-4** |

★ = masuk ke terminal state

#### (iii) Aksi Terbaik di Setiap State (Non-Terminal)

Aksi terbaik = $\arg\max_a Q(s, a)$.

**Tabel nilai Q final (hanya yang non-nol):**

| State | Q(N) | Q(E) | Q(S) | Q(W) | **Aksi Terbaik** |
|-------|------|------|------|------|------------------|
| (1,1) | 0 | 0 | 0 | 0 | Tie — semua 0 |
| (2,1) | 0 | **-4** | 0 | 0 | **N, S, atau W** (hindari E→Wumpus) |
| (1,2) | **-4** | 0 | 0 | 0 | **E, S, atau W** (hindari N→Pit) |
| (2,2) | 0 | **+4** | 0 | 0 | **E** → menuju Gold ✅ |
| (1,4) | 0 | 0 | 0 | 0 | Tie — belum dijelajahi |
| (2,3) | 0 | 0 | 0 | 0 | Tie — belum dijelajahi |
| ... | 0 | 0 | 0 | 0 | Tie — belum dijelajahi |

**Grid aksi terbaik (→ = E, ↑ = N, ↓ = S, ← = W):**

```
     x=1       x=2       x=3       x=4
y=4 | ?        | ?        |          | [PIT]    |
y=3 | ↓/→/←    | ?        | [PIT]    | ?        |
y=2 | ?        | →        | [GOLD]   | ?        |
y=1 | ?        | ↑/↓/←    | [WUMPUS] | ?        |
```

> [!insight] Interpretasi
> Setelah 3 episode, Q-learning baru belajar dari 3 jalur yang dieksplor. State yang belum dikunjungi (semua Q=0) belum memiliki preferensi aksi. Hanya 3 state yang mendapat update bermakna:
> - **(2,1):** Hindari E (menuju Wumpus, Q=-4)
> - **(1,2):** Hindari N (menuju Pit, Q=-4); E adalah arah menuju (2,2) yang dari sana bisa ke Gold
> - **(2,2):** **E** adalah aksi terbaik jelas (Q=+4), menuju Gold
